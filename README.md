# Todo List Application Documentation

## Features
- **Add Tasks**: Users can add new tasks to the list.
- **Delete Tasks**: Users can remove completed or unwanted tasks.
- **Edit Tasks**: Users can modify existing tasks.
- **Mark as Completed**: Users can mark tasks as completed, providing a visual distinction.
- **Local Storage Support**: The application stores tasks in the browser's local storage, allowing persistence across sessions.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/daffayuuu-ui/todo-list.git
   ```
2. Navigate to the project directory:
   ```bash
   cd todo-list
   ```
3. Open `index.html` in your browser.

## Usage
- Upon opening the application, users will see an input field to add tasks.
- After adding a task, it will appear in the list below.
- Users can check off tasks as completed, edit them, or delete them as needed.

## Local Storage
- The application utilizes the browser's local storage to save the tasks.
- Tasks remain available even after refreshing the page or closing the browser.
- This feature ensures that users do not lose their tasks between sessions.
